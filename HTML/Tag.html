Tag HTML

Un elemento HTML è definito da una tag iniziale (<tag>). Se l'elemento ha al suo interno altri contenuti, terminerà con un tag di chiusura (</tag>) in cui il nome del tag è preceduto da uno slash forward.

Abbiamo tre tipi di tag:

  • Block level tags: creano un elemento di tipo "block", cioè che occupa una intera linea, quindi obbliga gli elementi successivi ad andare su una nuova linea.

  • Inline tags: non obbligano gli elementi successivi ad andare su una nuova linea ma gli elementi rimangono sulla stessa linea.

  • Self closing tags: tag che non hanno bisogno di chiusura.

Un altra sottocategoria di tag è quella dei container, tag che servono a raggruppare un insieme di tag. Senza CSS non hanno una vera utilità se non quella di organizzare i tag.

Ai tag può essere aggiunto un attributo inserendo atttribute="value" all'intero del tag iniziale.


Lista dei tag

Block level tags:

  • Il tag <div>(division) definisce una divisione o una sezione in un documento HTML. È un tag container.

    Esempio:

    Questo è un div

  • Il tag <p>(paragraph) definisce un paragrafo. È un tag container.

    Esempio:

    Questo è un paragrafo

  • I tag da <h1> a <h6>(heading) definiscono intestazioni HTML.

    • <h1> definisce l'intestazione più importante, <h6> definisce l'intestazione meno importante.

    • Note d'uso:

      • Evita di utilizzare più di un <h1> per pagina, questo deve rappresentare l'intestazione principale per tutta la pagina.

      • Inoltre non saltare i livelli di intestazione: inizia con <h1>, poi <h2>, e così via.

      • Non utilizzare gli elementi di intestazione per ridimensionare il testo. Usa invece le proprietà di CSS.

    Esempio:

    Questa è un intestazione 1

    Questa è un intestazione 2

    Questa è un intestazione 3

    Questa è un intestazione 4

    Questa è un intestazione 5
    Questa è un intestazione 6

  • Il tag <ol>(ordered list) definisce una lista ordinata di elementi, tipicamente reso come un elenco numerato. Una lista ordinata può essere numerica o alfabetica.
    Il tag <li> viene utilizzato per definire ogni elemento della lista.

    Esempio:

    1. Fai il caffè
    2. Aggiungi il latte
    3. Aggiungi lo zucchero

  • Il tag <ul>(unordered list) definisce una lista non ordinata di elementi, tipicamente reso come una lista puntata.
    Il tag <li> viene utilizzato per definire ogni elemento della lista.

    Esempio:

    • Caffè
    • Latte
    • Zucchero

  • Il tag <li>(list item) definisce un elemento della lista.
    Il tag <li> viene utilizzato all'intero di <ol> e <ul>.

    Esempio:

    1. Caffè
    • Lattè

Inline tags:

  • Il tag <span>(span) definisce un container utilizzato per contrassegnare una parte del documento. È un tag container.

  • Il tag <a>(anchor tag) definisce un hyperlink, utilizzato per collegare da una pagina a un altra.
    L'attributo più importante del tag <a> è "href", che indica l'indirizzo di destinazione.

  • Il tag <button>(button) definisce un pulsante cliccabile.
    All'interno di un pulsante puoi inserire del testo e altri tag di stile.
    Ciò non è possibile con un pulsante create con l'elemento <input>.
    Consiglio: Specifica sempre l'attributo type del <button>, affinchè il tuo browser sappia che tipo di pulsante è.

  • Il tag <select>(select) va a creare un elenco a discesa, viene solitamente utilizzato nei form per collezionare i dati dell'utente.
    L'attributo name="" è necessario per fare riferimento ai dati del modulo dopo esser stato inviato (senza un attributo name="", i dati non verranno inviati).
    L'attributo id="" è necessario per associare l'elenco a discesa con un <label>.
    Gli elementi del tag <select> sono definiti dai tag <option>.
